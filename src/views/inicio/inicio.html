Bienvenido a WashTrack

Aquí puedes gestionar todo lo relacionado con tu lavadero: usuarios, tipos de vehículos, servicios, lavados y ver el historial de facturación. Selecciona una opción abajo para comenzar.

¿Qué puedes hacer?

Las acciones que puedes realizar depende del rol que tengas el cual contiene unos permisos especificos

  • Gestionar Usuarios
  • Gestionar vehiculos
  • Gestionar Tipos de vehiculos
  • Gestionar Servicios de vehiculos
  • Gestionar Tipos de lavados
  • Gestionar Items de lavados
  • Gestionar Gestionar lavados

Los elementos a los que no tengas permisos se negará el acceso.
